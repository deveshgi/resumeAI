# ResumeAI 🚀

An AI-powered Resume Builder and Resume Analyzer web application built using the MERN Stack. This project helps users create professional resumes with AI integration and modern UI design.

## 🌟 Features

* AI-powered resume generation
* Professional resume templates
* Responsive modern UI
* Resume preview and editing
* MERN Stack architecture
* REST API integration
* Backend authentication support
* Clean and user-friendly design

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/deveshgi/resumeAI.git
```

### Install Dependencies

```bash
npm install
```

### Run Frontend

```bash
npm start
```

### Run Backend

```bash
npm run server
```
